# Instruções permanentes deste projeto

Este projeto é um sistema de estudo.

Tudo o que for pedido aqui deve ser feito com a menor complexidade possível.

A prioridade é corrigir e ajustar apenas o front-end.

Não implementar regras de negócio, integrações, persistência, consultas, mutações ou qualquer outra parte de back-end, salvo pedido explícito.

Quando existir mais de uma forma de resolver, escolher sempre a opção mais simples, direta e fácil de manter.
